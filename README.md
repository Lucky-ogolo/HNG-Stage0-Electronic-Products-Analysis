# Electronic Products Analysis — HNG Stage 0

## Project Overview
This project was completed as part of the HNG Data Analytics Internship Stage 0 task. 
I explored a dataset of 10,000 electronic products to identify patterns in product 
availability, category distribution and brand performance using Excel.
---

![](https://github.com/Lucky-ogolo/HNG-Stage0-Electronic-Products-Analysis/blob/c2e02b24942649a34be75df0e0acd72ead0776cc/Stage0hng.png)
## Dataset
- 10,000 rows of electronic product data
- Columns include product name, brand, category, price, stock and availability status
- Source: HNG Internship provided dataset

## Questions Explored
1. What is the distribution of availability status across the product catalogue?
2. Which product categories have the highest number of products?
3. Which brands have the most products listed and what is their average price?

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Data Cleaning)

## Key Insights
- The availability status was fairly evenly distributed across all 6 categories
- Electronics and related categories dominated the product catalogue
- A small number of brands accounted for the majority of products listed
